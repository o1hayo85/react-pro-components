## 增加参数 Array 格式

增加 filterCascader、filterCheckbox、filterSelect、filterTreeSelect 的 isParamList 配置

### 解决的问题

参数字符串转 Array

### 带来的问题

- filterItems.params 类型不兼容(某项开启 isParamList 返回 Array,不开启返回 string)
- 只会影响直接从 filterItems.params 取某项值，且转类型

### 解决方案

- 字符串转 Array 的直接去掉,加上参数
- 将类型强转成 string
